Xcode快捷键

发表于 2015-06-04   |   分类于 Xcode

作为一名 ‘iOS开发工程师’，你应该学会熟练的使用XCode进行代码编写，所以建议大家去使用快捷键进行开发，因为：

  1. 可以让我们显得更加的专业，逼格更高
  2. 提高我们的开发效率，节省开发时间
  3. 使用任何人的电脑将熟练，快速的操作（不建议大家更改系统自带的快捷按键）

Xcode

快速创建文件

cmd + N :—> 创建文件 (Xcode 常用快捷键)

创建视图的时候在左侧的1区域和右侧的2区域切换的时候使用 ‘Tab’ 按键，这样就可以使用键盘利用上下左右按钮进行快速选择了

最常用

  1. cmd + A :—> 全选
  2. cmd + S :—> 保存
  3. cmd + X :—> 剪切
  4. cmd + V :—> 粘贴
  5. cmd + R :—> 运行
  6. cmd + B :—> 编译(并且可以保存所有文件哦)
  7. cmd + . :—> 停止运行
  8. cmd + shift + B :—> MRC下检测内存警告
  9. cmd + shift + K :—> 清理，按按更健康

各个区域的显示与隐藏

  1. cmd + shift + Y :—> 显示/隐藏底部的控制台（Xcode 常用快捷键）
  2. cmd + 0 :—> 显示/隐藏最左侧目录栏（Xcode 常用快捷键）
  3. cmd + alt + 0 :—> 显示/隐藏最右侧的工具栏（Xcode 常用快捷键）
  4. cmd + alt + enter / cmd + enter :—> 显示/隐藏Assistant Editor(在左侧、右侧同时显示一个类的.h、.m 文件)编辑视图（Xcode 常用快捷键）
  5. cmd + F :—> 显示搜索栏, ESC按钮隐藏
  6. cmd + alt + F :—> 显示替换栏, ESC按钮隐藏
  7. cmd + shift + F :—> 打开全局搜索，可以搜索项目中所有位置的信息

文件相关

  1. control + 5 :—> 输入类名，上下选择，快速跳转到该类
  2. control + 6 :—> 输入方法名，上下选择，快速跳转到方法
  3. cmd + L :—> 弹出输入框，输入数字，可快速跳转至该行
  4. cmd + shift + O :—> 快速跳转到文件，另外，按 cmd + shift + J 可以在目录中快速选中当前打开的文件（Xcode 常用快捷键）
  5. cmd + alt + 上/下 :—> .h、.m文件的切换
  6. cmd + alt + 左/右 :—> 显示的上一个文件和显示的下一个文件之间的切换，相当于触摸板的两个

代码编写

  1. cmd + 上 :—> 跳到当前文件的最上部
  2. cmd + 下 :—> 跳到当前文件的最下部
  3. cmd + 左 :—> 跳到当前行的最左侧
  4. cmd + 右 :—> 跳到当前行的最右侧
  5. esc (cmd + .) :—> 在敲代码的时候，如果需要提示，可以按esc隐藏/显示提示
  6. cmd + alt + [ / cmd + alt + ] :—> 将当前行或者选中的行进行上下行调整
  7. ctrl + i :—> 全选代码后，自动补齐

辅助功能

  1. cmd + shift + 0 :—> 打开api
  2. cmd + shift + 1 :—> 打开 Welcome To Xcode 页面
  3. cmd + shift + 3 :—> 全局截图（全局快捷键）
  4. cmd + shift + $ :—> 选择截图

触摸板的使用

  1. 两个手指的左右滑动，前后文件的查看
  2. 把光标放到方法上，三个手指轻拍触摸板，相当于按住alt去点击方法名，打开api简介

iOS Simulator

  1. cmd + 1 / cmd + 2 / cmd + 3 :—> 调整模拟器的大小
  2. cmd + 左/右 :—> 旋转模拟器
  3. cmd + S :—> 保存模拟器当前屏幕截图，保存到桌面上
  4. cmd + shift + H :—> Home按钮
  5. cmd + L :—> 锁屏
  6. cmd + shift + M :—> 模拟器内存警告
  7. cmd + T :—> 旋转的时候慢速动画的切换
  8. cmd + K :—> 显示/隐藏键盘 (需要出现跳动的光标)
