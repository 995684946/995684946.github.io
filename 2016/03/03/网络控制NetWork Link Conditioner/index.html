网络控制NetWork Link Conditioner

发表于 2016-03-03   |   分类于 Xcode

我们开发过程中可能需要在网络不好的情况下测试，我就介绍一个 Mac 和 iOS 的实用工具，用来精确和持续模拟不良的网络环境的软件：NetWork Link Conditioner，

安装

Network Link Conditioner 可以在 “Xcode 的硬件 IO 工具” 包中找到。

打开步骤：Xcode->Open Developer Tool->More Developer Tools

搜索“Hardware IO Tools”下载硬件工具包。NetWork Link Conditioner就在安装包内。

下载完成后，打开 DMG，然后双击 “Network Link Condition.prefPane” 来进行安装

现在起，你可以在系统设置的底部启用 Network Link Conditioner。

启用后，Network Link Conditioner 可以根据内置的某个预设来改变 iPhone 模拟器的网络环境根：

  • EDGE
  • 3G
  • DSL
  • WiFi
  • High Latency DNS
  • Very Bad Network
  • 100% Loss

每个预置可以设置上行或下行的带宽极限，延迟，和丢包概率（当任何值被设置为 0 时，该值将会同你的计算机的网络环境保持一致）。

注意

别忘了在完成测试后把它关掉！😓
